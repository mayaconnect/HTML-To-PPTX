    Known Limitations

    The converter cannot reproduce every CSS feature. Keep these in mind when generating slides:

    • Gradients (linear-gradient, radial-gradient) — use solid colors or overlapping shapes instead
    • FontAwesome icons — rendered as text in the browser, may appear as squares in PPTX if the font is not installed
    • SVG elements — inline SVGs are not supported; use images (PNG/JPG) instead
    • box-shadow, text-shadow, filters — not converted; use layered shapes to simulate shadows
    • CSS transforms (rotate, scale, skew) — not supported; position elements directly with left/top
    • Flexbox / Grid layouts — use position:absolute with explicit pixel values

    Tip: For best results, use the Structured Prompt and keep slides simple — one key idea per slide, solid colors, and clear text hierarchy.
